# Sommaire du guide utilisateur

Le guide utilisateur du plugin `panel-editor` est organisé en 3 parties :

- **Présentation**: [Vue globale](global_overview.md)
- **Configuration**: [Prendre en main](getting_started.md)
- **Restrictions par rôles**: [Barres ou menus d'outils](tools.md)

Le plugin affiche les entités identifiées sur la carte, permet de passer en mode édition et applique les règles de droits définies par couche.
